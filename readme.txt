Javagee One Heal One Earth Plugin Campaign against COVID-19 (Corona Virus) Global Pandemic 2020.

== Description ==
Javagee One Heal Plugin is a wordpress plugin that show users or sites to support the fight against COVID-19 (Corona Virus) Global Pandemic 2020. This plugin will show One Heal One Earth Logo within their website.


*HOW TO USE*
1.First Activate Plugin.
2.Go To Javagee One Heal One Earth Plugin Menu.
3.Use the default shortcode and paste anywhere in your posts/pages.

    [OneHeal]

4.To change image size.

    [OneHeal width=350px height=350px]


*PLUGIN MINIMUM REQUIREMENTS*
WordPress 3.1+
PHP 5.6+
MySQL 5.x


== Installation ==
1.Upload the onehealoneearth folder to the directory /wp-content/plugins/.
2.Activate the plugin using the ‘Plugins’ menu in WordPress.
